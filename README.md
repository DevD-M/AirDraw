# AirDraw

Draw on screen using hand gestures — no touch required.

## Tech Stack
- MediaPipe Hands (Google) — real-time hand landmark detection
- HTML5 Canvas API — drawing layer
- Vanilla JavaScript — gesture logic & DOM manipulation


## Features
- Index finger gesture — draw mode
- Pinch gesture — alternate draw mode  
- Clear canvas button
- Toggle landmarks on/off

## How to use
1. Clone the repo
2. Open `airdraw.html` in browser
3. Allow camera access
4. Point index finger to draw
